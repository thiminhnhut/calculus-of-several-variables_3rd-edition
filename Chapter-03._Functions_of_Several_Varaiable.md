# Functions of Several Variables

## Đường mức của hàm số nhiều biến

| Hàm số            | Đường mức   |
|-------------------|-------------|
| $x^2 + y^2$       | Hình tròn   |
| $ax^2 + by^2 = c$ | Hình elip   |
| $y-x^2$           | Parabol     |
| $xy$              | Hyperbol    |
| $ax+by$           | Đường thẳng |

- Cách vẽ: Chọn các giá trị mức c khác nhau, sau đó giải phương trình f(x,y) = c để tìm các đường cong mức.
